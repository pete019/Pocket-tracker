How to use

  1. Run the web server with npm install and npm start.
  2. Open the page in your browser at http://localhost:3000.
  3. Paste the M-Pesa SMS text and optionally the sender, then click Parse SMS and Lookup.
  4. Or search directly by parent name or phone number.
  5. If you want private sheet access, place a service account JSON at service_account.json.
